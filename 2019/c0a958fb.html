如何在windows系统下对iphone进行越狱并绕过ID锁
越狱 破解
越狱
发布日期:   2020-02-12
更新日期:   2020-02-12
文章字数:   2k
阅读时长:   7 分
阅读次数:  

该方法仅适用于自己手里记不住ID账号和密码的手机使用，请使用非法手段获取手机的伙伴将手机归还失主。

前言

手上有一个变砖很久的iphone，一直有ID锁，最近闲来没事，尝试了绕过ID锁，进行一番摸索，发现需要先越狱。越狱的话网上的教程有很多，一般情况下都是使用的越狱工具checkra1n，这个工具目前只有macOS版本的和linux版本的，win版还在开发过程中。对于没有mac电脑的伙伴而言，只能尝试黑苹果系统和linux版本的，推荐使用linux版本的，因为黑苹果系统很难安装，我在以前的电脑上尝试了一下电脑直接黑屏没有反应，最后在linux系统下越狱成功，在windows系统下进行绕过ID成功（绕过ID在任何系统下都可以进行，比较习惯于使用windows，所以在windows系统下操作）。

注意：不要使用虚拟机，貌似虚拟机不能成功越狱。

如果你有linux系统的电脑，直接在checkra1n.cn上下载linux版本的破解工具使用即可。如果没有linux系统的电脑，可以使用U盘安装一个带checkra1n的linux系统的镜像，不用装linux系统，插上U盘就可以使用，推荐使用这种方式。

还可以直接在电脑上装一个linux系统，我就是在电脑上安装了一个xubuntu系统。

废话不多说，直接上教程。

手机系统更新

手机系统必须要满足大于12.1.0（貌似是这个，记不清了）

工具

+U盘一个（不要小于2G）
+电脑一台
+工具包（简易系统ios，balenaEtcher-Portable-1.5.45）

工具在文章最后附件中下载。

对于被ID锁锁住的手机是不能直接使用手机更新系统的，必须依靠其他工具。手机更新推荐使用爱思助手，可以一键下载最新系统并进行更新。

手机系统更新步骤

1.下载爱思助手，这个步骤很简单，下载安装即可

2.手机连接电脑，爱思助手会自动识别

3.点击刷机越狱

4.下载系统版本（注意，这里需要重点说明：不要下载12.4.5版本的系统，这个版本是绕不过ID锁的，苹果公司最近进行过升级，已经不能绕过了，所以使用12.4.4。我最开始就是下载的12.4.5，无论怎么都绕不过去，最后尝试了12.4.4就成功了 ）

5.系统下载完成之后，直接点击立即刷机就行了

6.接下来就是等待刷机完成，整个刷机过程中请保证设备与电脑连接正常稳定，使用一根好的数据线。

越狱

越狱是必须要进行的一步，使用checkra1n进行越狱

U盘安装系统越狱

现在说一说如何使用u盘安装一个带有checkra1n工具的linux镜像系统并进行越狱。

操作步骤如下：

1.电脑插入U盘

2.打开balenaEtcher-Portable-1.5.45.exe

3.选择 Select image ，选择bootra1n-x86_64-0.9.8-20200206.iso，选择你的U盘，再点Flash

4.制作完成后，U盘拔掉，重新插一次

5.重启电脑，修改启动项用U盘启动，这个根据自己的电脑自行配置，不同的电脑的配置方式不一样，我使用的thinkpad是按F12可以选择启动方式，直接选择U盘启动就行了（部分电脑需要进入BIOS修改启动项）

选择第一个，按enter进入系统，没有图形界面，是命令行操作界面。

6.进入终端后，输入anon 回车，密码 voidlinux（不显示密码） 回车

7.输入 sudo checkra1n 回车（注意！部分电脑1字无法输入，需要英文上面1键才行）

8.如下图已经启动了checkra1n，点击[ Start ]。

上述界面上，请注意黄色部分字的提示。
因为我的手机好几年没有使用，系统没有进行更新，所以提示iphone6是支持的，但是系统不支持，并且会提示支持的系统版本，这样就需要对手机系统进行更新了（更新到12.1版本以上）。
手机系统更新教程在前面第二部分。

9.如下进入recovery模式

Iphone显示显示进入如下界面

10.进入DUF模式，按照下述步骤，先点击[ Start ]，之后同时按住手机的电源键和Home键持续4s，之后松开电源键，保持按住Home键10s

11.Checkra1n提示进入了DUF模式，松手，等待完成 all done表示已经完成。进行完上述步骤之后，等待越狱完成即可，如果失败了，请按照步骤再试一次。

Linux系统运行checkra1n方法

如果有linux系统，到checkra1n官网去下载对应的软件，然后打开终端，输入以下命令（来自官网）：

**

  1. echo “deb https://assets.checkra.in/debian /“ | sudo tee -a /etc/apt/sources.list
  2. sudo apt-key adv –fetch-keys https://assets.checkra.in/debian/archive.key
  3. sudo apt updade
  4. 进入下载checkra1n软件的文件夹，输入下面的命令
  5. Chmod 777 checkra1n
    6 ./checkra1n
  6. 软件开始运行，按照提示完成即可。

**

绕过ID锁

由于手头的手机还有ID锁，就算是越狱完成，还是进不了系统，这就需要使用特殊的方法进行绕过了，接下来使用的方法也是在网上搜索而来，但是一般都是使用的MAC os系统来完成了，但其实任何系统下都可以进行，需要使用一款ssh工具即可，我这里选用putty。

绕过ID锁步骤

下面就说步骤：

1.打开爱思助手

2.点击工具箱，选择“打开SSH通道”（如果提示无法打开，就是没有越狱，请先越狱）

3.会弹出iphone手机ssh通信的ip地址，用户名与密码以及端口信息，请记住

4.打开putty工具

5.在Host Name输入127.0.0.1（IP地址），Port下输入你的端口，我的是22，点击Open，到弹出的窗口中输入如下信息

6.输入用户名：root

7.输入密码：alpine

8.依次输入下面的命令：

9.“mount -o rw,union,update /”并回车

10.“echo “” >> /.mount_rw”并回车

11.“mv /Applications/Setup.app /Applications/Setup.app.crae”并回车

12.“uicache –all”并回车

13.“killall backboardd”并回车，手机会重启，之后正常进入系统

使用工具绕过

如果使用上面的方法无法绕过，还可以使用windows下的一款工具iCloudBypass ShiftKey
使用该工具也需要先越狱，打开ssh端口，原理与上面的命令类似。

通过上述方式，我的手机已经成功绕过ID锁了，美滋滋。

关于绕过ID锁，有以下需要注意的点：

1.绕过激活锁不代表退出已登录的 Apple ID，如果重新刷机或者抹除设备需要重新激活，所以绕过之后不要再刷新系统了。
2.绕过激活锁后无法通话、使用移动网络，但是 WiFi 功能可以正常使用。
3.如果需要下载应用，需要注册一个苹果用户，然后使用你注册的用户下载软件并安装即可，手机上如果注册不了，可以通过电脑使用网页注册。

总结

简单的来说ID是绕过去了，但是手机功能相当于一部iPad。除了接打电话不能使用以外其他功能是正常的。

使用以上方式，我的吃灰iphone6已经复活了，下载软件没有问题，可以充当一个看电视和玩游戏的手机，哈哈。
